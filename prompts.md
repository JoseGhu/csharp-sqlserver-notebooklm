# Engenharia de Prompts

## Prompt 1

**Pergunta:**
Explique como funciona a conexão entre C# e SQL Server.

**Resultado obtido:**
Foi apresentada uma visão geral sobre connection strings e comunicação entre aplicação e banco de dados.

---

## Prompt 2

**Pergunta:**
Mostre um exemplo de CRUD em C# utilizando SQL Server com ADO.NET.

**Resultado obtido:**
Foi gerado um exemplo utilizando SqlConnection, SqlCommand e SqlDataReader.

---

## Prompt 3

**Pergunta:**
Crie um exemplo de API em C# utilizando Entity Framework Core e SQL Server.

**Resultado obtido:**
Foi apresentada uma estrutura contendo Model, DbContext e Controller.

---

## Prompt 4

**Pergunta:**
Qual a diferença entre ADO.NET e Entity Framework Core?

**Resultado obtido:**
Foi possível compreender as vantagens e características de cada abordagem.

---

## Dificuldades Encontradas

* Respostas muito genéricas quando os prompts eram pouco específicos.
* Necessidade de pedir exemplos completos para obter respostas mais práticas.
* Dúvidas iniciais sobre as diferenças entre ADO.NET e Entity Framework Core.

## Conclusão

A utilização de prompts mais específicos permitiu obter respostas mais detalhadas e exemplos mais próximos de aplicações reais em C# e SQL Server.
